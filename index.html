NotPly is a simple plugin that let's you show notifications, There are 3 types of notifications here:

To start The notply plugin on the element you want make NotPlyInitiate("element"), Example : NotPlyInitiate("body") or NotPlyInitiate("#bigElement")

Notifications :

Notifications are the basic plugins in here, also you can activate notifications with data attributes or with the notply() function, there are many notply() functions so be careful which one you want, but simply you can use the notply("the notification") function, if you want to make the data attribute function, see the demo

Toasts :

Toasts Are little notifications that appear in any position of the screen, they can be used with the notToast() function, the full functionality of this function will be:

notToast({
text : "text",
title : "title",
theme : "theme",
position : "position",
transition : "second",
Doctitle : true / false,
});

Alerts :

Alers are Here to replace the default javascript popup window, they are available only with the alertAs() function, also this function have more functions like alertAs("text","title"), alertAsPrompt("text","title","submit button","onsubmit","oncancel"), alertAsConfirm("text","title","submit button","onsubmit","oncancel"), alertAsLoading("text", delay, "alert") If you want to make a get prompt box value, make popUpInput()
Example:
alertAsPrompt("Hi","Hello","Go","alertAs('Submitted as : ' + popUpInput())","alertAs('Canceled')")
