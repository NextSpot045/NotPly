NotPly is a simple plugin that let's you show notifications, There are 3 types of notifications here:

Hey Hey, and also, to use this plugin, use $('body').NotPly() But make sure it's on a big container like <html> tag or <body> tag, $('#bigElement').NotPly(), if you want use the $('body').NotPly() method,

Notifications :

Notifications are the basic plugins in here, also you can activate notifications with the $('body').NotPly() method, simply you can use the $('body').NotPly({
text: "<p>Hello </p>",
oncancel: function(){
alert("Cancel")
},
type: "notply",
icon: "fa fa-file",
theme: "notply-warning",
transition: ".8s"
}); method, if you want to make the data attribute method, see the demo

Toasts :

Toasts Are little notifications that appear in any position of the screen, they can be used with the $('body').NotPly() method, the full functionality of this method will be:

$('body').NotPly({
text: "<p>Hello </p>",
title: "Title",
oncancel: function(){
alert("Cancel")
},
type: "toast",//Type
theme: "notply-warning",
transition: ".8s"
});

Alerts :

Alers are Here to replace the default javascript popup window, they are available only with the $('body').NotPly() method, also this method have more methods like $('body').NotPly({
text: "<p>Hello </p>",
title: "Title",
Sbuttons: ["hello",'hii'],
oncancel: function(){
alert("Cancel")
},
type: "alert",
theme: "info",
transition: ".8s"
});

Sheets :

Sheets are bottom aligned modals, they can be called with the $('body').NotPly(), To call them use $('body').NotPly({
text: "<p>Hello </p>",
title: "Title",
oncancel: function(){
alert("Cancel")
},
type: "sheet",
theme: "custom",
transition: ".8s"
});
